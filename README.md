# 🌍 Air Pollution Data Analysis & Visualization

## 📌 Project Overview

This project focuses on analyzing air pollution data to uncover meaningful insights and patterns using **data analysis and visualization techniques**. It also includes a simple **machine learning model** to predict pollution levels.

---

## 🎯 Objectives

* Identify the most polluted cities
* Compare pollution levels across states
* Analyze different pollutant types
* Study the relationship between pollution indicators
* Understand the distribution of pollution levels

---

## 📂 Dataset

The dataset contains information about:

* State and City
* Pollutant Type (PM2.5, NO2, etc.)
* Minimum, Maximum, and Average Pollution Levels
* Date and Time of Data Collection

---

## 🛠️ Tools & Technologies

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

---

## 🔍 Exploratory Data Analysis (EDA)

Performed detailed EDA to:

* Handle missing values
* Remove duplicates
* Understand data distribution
* Identify trends and relationships

### 📊 Visualizations:

* Bar Charts (Top polluted cities & states)
* Pie Chart (Pollutant contribution)
* Heatmap (Correlation analysis)
* Histogram (Distribution of pollution levels)

---

## 🤖 Machine Learning Model

* Model Used: **Linear Regression**
* Features: `pollutant_min`, `pollutant_max`
* Target: `pollutant_avg`

### 📈 Evaluation:

* Mean Squared Error (MSE) used to evaluate performance

---

## 📊 Key Insights

* Some cities show significantly higher pollution levels
* Pollution varies across different states
* Certain pollutants contribute more to overall pollution
* Strong correlation exists between min, max, and average pollution values
